Sinatra Gossip
===============

Ceci est un repo de vilains gossips

Pour lancer le serveur des gossip : shotgun -p 4567

Puis rdv sur : http://localhost:4567/

## config.ru
Permet de lancer le programme

## controller.rb
La class ApplicationController update et charge les différentes pages 

## Gossip.rb
Les différents méthodes pour mettre les potins dans le csv, les rechercher...

## dossier Views 
Les différentes page 

1. index : page principale
2. new_gossip : page pour créer un nouveau gossip
3. show : page pour un gossip
4. edit : page pour modifier un gossip
